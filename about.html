ABOUT THIS PROJECT

My First HTML Website Experience

This is my first attempt at building a multi-page website using only HTML. I just learned the basics of HTML tags like headings, paragraphs, links, and tables. The most challenging part was figuring out how to embed videos, which led me to discover iframe tags.

What I Built

I created a 6-page educational website about web development fundamentals without using any CSS or JavaScript. The entire structure relies on HTML tables for layout and basic HTML attributes for formatting.

Learning Process

I used reference materials and AI assistance to create the educational content for each topic. The notes you see on HTML basics, forms, tables, and internet concepts were compiled from various sources and then organized into a beginner-friendly format.

What Worked Well for a Beginner

  • Table-based layout gave me control over page structure
  • Consistent navigation across all pages
  • Successfully embedded video content using iframes
  • Created working internal links between pages
  • Organized content in a logical learning sequence

Challenges and Drawbacks

  • Repetitive Code: I had to copy the same table structure and navigation to every page
  • Limited Styling: Without CSS, the website looks very basic and dated
  • Table Layouts: Using tables for layout is not the modern approach
  • Accessibility Issues: The structure could be better for screen readers
  • Mobile Unfriendly: Fixed table widths don't work well on small screens
  • Maintenance: Making changes requires editing multiple files

What I Learned

This project taught me the fundamentals of HTML structure, file organization, and how web pages connect together. I discovered how much work goes into even a simple website and gained appreciation for CSS and modern web development tools.

While the approach has limitations, it was a valuable learning exercise that helped me understand basic web development concepts from the ground up.I built It in total 3 session each of 2 to 3 hours(In total 9 hours).Used AI for Notes and faster development but the idea and all the structure all are original and belongs to me . I feeling very confident after finishing it.

Built with determination, basic HTML knowledge, and lots of trial and error!
