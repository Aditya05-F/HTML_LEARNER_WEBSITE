HTML : Hyper Text Markup Language

HTML stands for HyperText Markup Language. It is the standard language used to create and structure the content of a webpage.

HyperText

This refers to the ability to create links that connect web pages to one another, making the web a connected "web" of information.

Markup Language

This means you use "tags" to surround your content, giving that content meaning and structure. You are "marking up" a plain text document.

1. The Heading (<h1> to <h6>)

What it is: Headings are tags used to define titles and subtitles on your page. They are crucial for creating a logical hierarchy and outline for your content.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
  • <h1>: The most important heading, typically used only once per page.
  • <h2>: A major section heading.
  • <h3>: A sub-section heading under an <h2>.
  • ...and so on down to <h6>.

The Purpose: To structure your document logically for humans and search engines. It is not just for making text big; it gives the text structural importance.

2. The Paragraph (<p>)

What it is: The most common tag used to group sentences and blocks of text together.

The Tag: <p>

The Purpose: Defines a separate paragraph. Browsers automatically add space before and after a <p> element.

<p>This is the first paragraph. It contains several sentences about a topic.</p>

<p>This is a second, separate paragraph. The browser will display it with a space between it and the first one.</p>

3. HTML Horizontal Rules

The <hr> tag defines a thematic break in an HTML page, usually rendered as a horizontal line.

<p>This is the first topic.</p>
<hr />
<p>This is a completely different topic.</p>

4. The Line Break Tag: <br>

This tag tells the browser to break the line and continue writing on the next line.

Purpose: Creates a single line break.

<p>221B Baker Street<br>
London, England</p>

5. The Anchor Tag (<a>)

What it is: The anchor tag creates hyperlinks to other pages, files, or within the same document.

Tag: <a>

Purpose: Makes text or image clickable. Requires an href attribute (hypertext reference).

<a href="https://www.coderarmy.in/">Visit Coder Army</a>

HTML Links - The target Attribute

By default: links open in the same browser tab.

target values:

  • _self - Default, opens in same tab.
  • _blank - Opens in a new tab or window.

6. The Image Tag (<img>)

What it is: The Image tag is self-closing and used to embed images.

Purpose: To display a visual image. Requires attributes:

  • src: Path or URL to the image file.
  • alt: Alternative text (for accessibility and fallback).
