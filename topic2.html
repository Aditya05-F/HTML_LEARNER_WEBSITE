HTML Nested Lists & Tables

HTML Lists and Tables

This page provides information about how to use lists, nested lists, tables, and nested tables in HTML. It's intended for beginners learning HTML structure.

1. Lists in HTML

Lists are used to organize content in an ordered or unordered manner.

1.1 Unordered List (<ul>)

An unordered list displays items with bullet points.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
  • Item 1
  • Item 2

1.2 Ordered List (<ol>)

An ordered list displays items with numbers.

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>
  1. First item
  2. Second item

1.3 Nested Lists

You can nest one list inside another for hierarchical data.

<ul>
    <li>Fruits
        <ul>
            <li>Apple</li>
            <li>Banana</li>
        </ul>
    </li>
    <li>Vegetables
        <ul>
            <li>Carrot</li>
            <li>Peas</li>
        </ul>
    </li>
</ul>
  • Fruits
    • Apple
    • Banana
  • Vegetables
    • Carrot
    • Peas

2. Tables in HTML

Tables are used to display data in rows and columns.

2.1 Simple Table

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>25</td>
    </tr>
    <tr>
        <td>Bob</td>
        <td>30</td>
    </tr>
</table>
Name Age
Alice 25
Bob 30

2.2 Nested Tables

You can embed a table inside another table cell to represent complex data.

<table>
    <tr>
        <th>Department</th>
        <th>Employees</th>
    </tr>
    <tr>
        <td>HR</td>
        <td>
            <table>
                <tr>
                    <td>Alice</td>
                    <td>Manager</td>
                </tr>
                <tr>
                    <td>John</td>
                    <td>Assistant</td>
                </tr>
            </table>
        </td>
    </tr>
</table>
Department Employees
HR
Alice Manager
John Assistant

Summary

  • Lists: Help organize content using <ul> or <ol>.
  • Nested Lists: Useful for representing hierarchical data.
  • Tables: Allow displaying structured data in rows and columns.
  • Nested Tables: Useful for representing complex relationships within table cells.

This tutorial covers the basic structure and usage of lists and tables in HTML to help learners start building structured web pages.
